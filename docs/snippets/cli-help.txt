Bckupr is a tool to manage backups  your docker volumes and filesystem.
This application automates creating new backups and restoring data.

Usage:
  bckupr [command]

Backup Commands:
  backup      Create new backup
  delete      Delete backup
  list        List backups
  restore     Restore from backup

Daemons Commands:
  cron        Scheduled backups
  daemon      Cron/Web daemon

Additional Commands:
  completion  Generate the autocompletion script for the specified shell
  debug       Debug Bckupr
  help        Help about any command
  version     Print version

Flags:
      --debug     debug (default true)
      --dry-run   dry-run (default true)
  -h, --help      help for bckupr

Use "bckupr [command] --help" for more information about a command.
